# Transformer Exploration

## Overview

Project goes into detail a transformer that is trained on Project Hail Mary's story (refer to input.txt). The overall goal is to explore each component of the architecture and overall training in depth while detailing tradeoffs, ideas for optimization, new novel ideas, etc. - this is done in the form of Markdown files 

## Getting Started

### Prerequisites

- Ensure device has GPU related functionalities (DO NOT RUN ON CPU)

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd Transformer

# Install dependencies
pip install -r requirements.txt
```

### Usage

```bash
python main.py
```
